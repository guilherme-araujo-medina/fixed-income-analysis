# Fixed Income Analysis

This repository contains studies and applications related to fixed income analysis using Python.

## Objective

The goal of this project is to apply basic fixed income concepts such as bond pricing, duration and convexity in a structured and practical way.

## Topics covered

- Bond pricing
- Yield
- Duration
- Convexity
- Interest rate sensitivity
- Fixed income analysis

## Tools used

- Python
- numpy
- pandas
- matplotlib

## Project status

This repository is currently being developed as part of my learning journey in quantitative finance and financial analysis.
